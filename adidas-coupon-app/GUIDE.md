## 1. 설치

### 1.1 필수 요구사항

- **운영체제**: Windows 10 이상
- **Chrome 브라우저**: 최신 버전 권장
- **인터넷 연결**: 필수

### 1.2 프로그램 설치

1. `아디다스쿠폰발급프로그램_데이터.exe` 파일을 다운로드합니다.
2. 다운로드한 파일을 더블클릭하여 실행합니다.
3. 설치 경로를 선택하고 설치를 완료합니다.

### 1.3 웹크롤러 설치

프로그램 첫 실행 시 웹크롤러 설치가 필요합니다.

1. 프로그램 실행 후 우측 상단의 **⚙ 버튼**을 클릭합니다.
2. **"웹크롤러 프로그램"** 옆의 **[설치]** 버튼을 클릭합니다.
3. 설치가 완료되면 버튼이 **[설치 완료]**로 변경됩니다.

### 2.2 계정 등록

#### 단일 계정 등록

1. 좌측 하단의 **[+ 계정 추가]** 버튼을 클릭합니다.
2. 이메일과 비밀번호를 입력합니다.
3. **[저장]** 버튼을 클릭합니다.

<!--
📸 스크린샷 위치: 계정 추가 모달
[계정 추가 모달 캡처 이미지]
- 빨간 박스 1: 이메일 입력 필드
- 빨간 박스 2: 비밀번호 입력 필드
- 파란 박스: 저장 버튼
-->

#### 일괄 계정 등록

1. 좌측 하단의 **[📋 일괄 등록]** 버튼을 클릭합니다.
2. 텍스트 영역에 계정 정보를 입력합니다.
   - 형식: `이메일:비밀번호` (한 줄에 하나씩)
   - 예시:
     ```
     user1@example.com:password1
     user2@example.com:password2
     ```
3. **[등록]** 버튼을 클릭합니다.

### 2.3 정보 조회

계정의 포인트, 쿠폰, 바코드 등의 정보를 조회합니다.

#### 단일 조회

1. 조회할 계정의 행에서 **[정보 조회]** 버튼을 클릭합니다.
2. Chrome 브라우저가 자동으로 열리고 로그인이 진행됩니다.
3. 조회가 완료되면 테이블에 정보가 업데이트됩니다.

<!--
📸 스크린샷 위치: 단일 정보 조회 버튼
[테이블 행 캡처 이미지]
- 빨간 박스: 정보 조회 버튼
-->

#### 일괄 조회

1. 조회할 계정들의 **체크박스**를 선택합니다.
   - 전체 선택: 헤더의 체크박스 클릭
2. 상단의 **[📊 선택 정보조회]** 버튼을 클릭합니다.
3. 선택한 계정들이 순차적으로 조회됩니다.

<!--
📸 스크린샷 위치: 일괄 조회
[테이블 상단 캡처 이미지]
- 빨간 박스 1: 체크박스들
- 빨간 박스 2: 선택 정보조회 버튼

> ⚠️ **주의사항**
> - 일괄 조회 시 계정당 약 15~30초 소요됩니다.
> - 조회 중에는 Chrome 창을 닫지 마세요.

### 2.4 쿠폰 발급

적립된 포인트로 쿠폰을 발급합니다.

#### 쿠폰 종류

| 쿠폰 | 필요 포인트 | 설명 |
|------|-------------|------|
| 네이버페이 5천원 | 5,000P | 네이버페이 포인트 전환 |
| 스타벅스 5천원 | 5,000P | 스타벅스 기프티콘 |
| 10만원 쿠폰 | 100,000P | 아디다스 매장 사용 |

#### 단일 발급

1. 계정 행의 **[쿠폰 발급]** 버튼을 클릭합니다.
2. 발급할 쿠폰 종류를 선택합니다.
3. 발급이 완료되면 쿠폰 목록이 업데이트됩니다.

<!--
📸 스크린샷 위치: 쿠폰 발급 버튼 및 모달
[쿠폰 발급 모달 캡처 이미지]
- 빨간 박스 1: 쿠폰 발급 버튼
- 빨간 박스 2: 쿠폰 종류 선택
-->

#### 일괄 발급

1. 발급할 계정들을 체크박스로 선택합니다.
2. 상단의 **[🎁 선택 쿠폰발급]** 버튼을 클릭합니다.
3. 쿠폰 종류를 선택하고 발급을 시작합니다.

### 2.5 바코드 다운로드

계정의 멤버십 바코드 이미지를 다운로드합니다.

#### 단일 다운로드

1. 계정 행의 바코드 열에서 **[다운로드]** 아이콘을 클릭합니다.
2. 바코드 이미지가 자동으로 다운로드됩니다.

<!--
📸 스크린샷 위치: 바코드 다운로드
[바코드 열 캡처 이미지]
- 빨간 박스: 다운로드 아이콘
-->

#### 일괄 다운로드

1. 다운로드할 계정들을 선택합니다.
2. 상단의 **[📥 바코드 다운로드]** 버튼을 클릭합니다.
3. 선택한 계정들의 바코드가 ZIP 파일로 다운로드됩니다.

### 2.6 필터링

다양한 조건으로 계정을 필터링할 수 있습니다.

#### 필터 옵션

| 필터 | 설명 |
|------|------|
| **포인트 범위** | 최소~최대 포인트 지정 |
| **생일 월** | 특정 월에 생일인 계정 |
| **쿠폰 종류** | 특정 쿠폰 보유 계정 |
| **쿠폰 유무** | 쿠폰 있음/없음 |
| **계정 상태** | 활성/비활성 |
| **조회 상태** | 조회완료/대기중/오류 등 |

#### 필터 사용법

1. 테이블 상단의 **[🔍 필터]** 버튼을 클릭합니다.
2. 원하는 필터 조건을 설정합니다.
3. **[적용]** 버튼을 클릭하면 필터가 적용됩니다.
4. 필터를 해제하려면 **[초기화]** 버튼을 클릭합니다.

### 2.7 통계

상단 통계 카드에서 계정 현황을 한눈에 확인할 수 있습니다.

| 통계 카드 | 설명 |
|-----------|------|
| **계정** | 활성/전체 계정 수 |
| **만료 예정 쿠폰** | 7일 이내 만료 쿠폰 보유 계정 (클릭하면 필터 적용) |
| **10만원 쿠폰** | 10만원 쿠폰 보유 계정 (클릭하면 필터 적용) |
| **총 포인트** | 모든 계정의 포인트 합계 |

## 문제 해결

### Chrome이 열리지 않는 경우

1. Chrome이 설치되어 있는지 확인합니다.
2. 웹크롤러를 다시 설치합니다. (⚙ → 웹크롤러 프로그램 → 설치)

### 로그인이 실패하는 경우

1. 이메일/비밀번호가 정확한지 확인합니다.
2. 아디다스 웹사이트에서 직접 로그인이 되는지 확인합니다.
3. 계정이 비활성화되었을 수 있습니다.

### 조회가 느린 경우

- 인터넷 연결 상태를 확인합니다.
- 아디다스 서버 상태에 따라 지연될 수 있습니다.
